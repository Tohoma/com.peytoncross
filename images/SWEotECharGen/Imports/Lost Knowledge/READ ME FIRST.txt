Lost Knowledge
--------------

This folder contains adversaries and encounters that can be used with Lost Knowledge, the adventure that comes with the Force and Destiny Beta Rulebook.  This folder also contains a data set that can be imported to give you the new lightsaber crystal described in this adventure.

To include these new items in your system:

1. Run the data editor and Import the data set found in the "Lost Knowledge Imports\Data Sets" folder.  This will add the new lightsaber crystal attachment to your custom data.
2. Run the GM Tools.
3. Import all adversaries found in the "Lost Knowledge Imports\Adversaries" folder (use Import from the Adversaries pane).
4. Import the encounters found in the "Lost Knowledge Imports\Encounters" folder (use Import from the Encounters pane).
