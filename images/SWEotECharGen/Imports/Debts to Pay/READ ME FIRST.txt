Debts to Pay
------------

This folder contains adversaries and encounters that can be used with Debts to Pay, the adventure that comes with the Edge of the Empire GM's Kit.

To include these new items in your system:

1. Run the GM Tools.
2. Import all adversaries found in the "Debts to Pay Import\Adversaries" folder (use Import from the Adversaries pane).
3. Import the encounters found in the "Debts to Pay Import\Encounters" folder (use Import from the Encounters pane).
